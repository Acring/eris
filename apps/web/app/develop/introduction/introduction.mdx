# Eris UI

`eris-ui` 是一个基于 `React` + `Radix UI` + `Tailwind` 的 UI 组件库。用于快速构建 XSKY 内部产品，提高产品的一致性和开发效率。

---

## 特性

- 基于 `React` + `Radix UI` + `Tailwind` 的 UI 组件库
- 使用 `TypeScript` 编写，提供完整的类型定义文件
- 使用 npm + Git 管理，方便更新和维护
- 使用 Storybook 作为开发环境，方便开发和测试
- 不使用任何第三方组件库，减少打包体积
- 使用 turborepo 管理多个包，方便开发和维护，引入缓存机制，提高构建速度
