# 概览

随着 XSKY 产品线的丰富，越来越多的产品被开发出来，但是在早期开发中，为了产品的快速迭代，我们往往会忽略一些基础的设计，导致后期产品的维护成本越来越高，甚至无法维护。

例如 SDDC 和 SDS 作为 XSKY 产品线的核心产品，其设计原则并没有得到统一，虽然暂时没有影响，但是随着产品的迭代，这些问题会逐渐暴露出来。例如用户会觉得 SDDC 和 SDS 的产品逻辑不一致，或者在使用中会遇到一些奇怪的问题。

为了解决这个问题，我们需要统一产品的设计原则，这样才能保证产品的一致性，也能够降低产品的维护和迭代成本。
